Syntax Parsers:

A program that reads your code and determines what it does and if its grammar is valid. Your code isn't magic. Someone else wrote a program to translate it for the computer.

Lexical Environment:

Where something sits physically in the code you write. A lexical environment exists is programming languages in which where you write something is important.

Execution Context:

A wrapper to help manage the code that is running. There are lots of lexical environments. Which one is currently running is managed via execution contexts. It can contain things beyong what you've written in your code.
